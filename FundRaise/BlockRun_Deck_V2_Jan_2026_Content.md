- **Headers:** Sans-serif, bold, 48-60pt
- **Body Text:** Sans-serif, regular, 24-32pt
- **Data/Numbers:** Monospace or bold sans-serif, 36-48pt
- **Footnotes:** Sans-serif, light, 16-20pt

### Layout Principles
1. **Minimal text per slide** - investors should listen to you, not read slides
2. **High contrast** - ensure readability in any lighting
3. **Consistent spacing** - maintain visual hierarchy
4. **Data visualization** - use tables, diagrams, charts where possible
5. **Whitespace** - don't crowd slides; let content breathe

### Icon Style
- Simple, outline-based icons
- Consistent stroke width (2-3px)
- Monochromatic or using primary blue
- 48-64px size for inline icons
